# QR Code Generator & Scanner Using Python

## Project Overview

QR Code Generator & Scanner is a Python automation project that allows users to create QR codes from text or URLs and scan QR codes to extract stored information.

The project demonstrates automation using Python libraries for QR code generation and image-based scanning.

## Objectives

- Generate QR codes automatically
- Scan and decode QR code information
- Automate data sharing through QR technology
- Build a simple Python-based utility application

## Technologies Used

- Python
- QR Code Libraries
- Image Processing
- Automation

## Features

- Generate QR codes from text and URLs
- Scan QR code images
- Extract encoded information
- Save generated QR codes
- Simple user interaction

## Project Workflow

1. Accept user input
2. Generate QR code image
3. Save QR code output
4. Scan QR code image
5. Decode and display information
